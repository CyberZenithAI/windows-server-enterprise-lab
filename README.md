# Windows Server 2016 Enterprise Lab

Laboratorio de infraestructura corporativa con Windows Server 2016. Incluye dominio Active Directory, políticas GPO avanzadas, DNS/DHCP, servidor FTP seguro y auditoría de eventos.  

## Objetivos
- Configurar dominio corporativo con usuarios y grupos
- Aplicar GPO para hardening de estaciones Windows 10
- Implementar servicios críticos (DNS, DHCP, FTP seguro)
- Practicar auditoría y gestión de seguridad en Active Directory

## Tecnologías
- Windows Server 2016
- Active Directory
- Group Policy Objects (GPO)
- Windows 10 Clients
